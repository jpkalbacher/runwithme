- [ ] Create an account
- [ ] Log in / Log out
- [ ] Create Event

  - [ ] start time and end time
  - [ ] event name
- [ ] Edit/Delete event
- [ ] As a user I should be able to view all of my upcoming activities on the map
- [ ] Search for friends
- [ ] Add friends
- [ ] View all of friends upcoming activities on map
